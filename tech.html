Common Network Troubleshooting Commands

  • ping: uses ICMP - sends echo packet and listens

  • tracert: route and response of packes

  • ipconfig: shows config of host IP both static and dynamic

  • nslookup: DNS resolution, finds IPs in DNS

  • netstat: state of connections on host

  • Putty/TeraTerm: connect to different systems; telnet, SSH, serial client

  • subnet IP calculator: ensure correct IP address selection

  • speedtest.net: quick test determine bandwidth and quality

  • pathing/mtr: conbines tracert + ping

  • route: displays status of routing table on host; used with multiple IP/Gateway
